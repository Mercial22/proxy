## 方法二：系统命令或设置（无需联网）

### Windows系统

#### 命令提示符查询

1.  按 **Win + R** 打开“运行”，输入 `cmd` 并回车；
    
2.  在命令行中输入 `ipconfig` 并回车；
    
3.  查找输出中的 **IPv4 地址** 即可获取本机局域网IP。
    



#### 网络设置查看

1.  打开 **控制面板** → **网络和共享中心**；
    
2.  查看当前连接的网络 → 点击 **详细信息**；
    
3.  查找 **IPv4 地址**。
    

### macOS系统

1.  打开 **系统偏好设置** → **网络**；
    
2.  选择当前网络接口 → 点击 **高级**；
    
3.  在 **TCP/IP** 选项卡即可查看IP地址。
    

### 优势

*   **无需联网**：适合局域网环境下查看IP；
    
*   **信息精确**：可获取IP、子网掩码、网关等网络参数；
    
*   **便于技术管理**：适合店铺运营团队进行多设备配置和网络排查。
